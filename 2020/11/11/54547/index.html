On accurate POI recommendation via transfer learning

Posted on 2020-11-11 In UrbanComputing

文章主要解决什么问题

POI 推荐中的稀疏与冷启动问题。

解决的思路是什么

使用 Transfer Learning 的方法，融合了有价值的跨域知识，实现了更准确的POI推荐。捕获了其他领域的用户交互，并将有用的偏好引入POI推荐，可以解决数据稀疏性问题。
与基于矩阵分解的跨域技术相比，该方法采用深度迁移学习，能够学习复杂的用户-项目交互关系，准确捕捉用户的总体偏好进行迁移。

核心的知识点有哪些

  • Transfer Learning

模型分块说明

  1. embedding 模块
    对 user 以及 source data 使用 one-hot 编码再 embedding。target data 使用 一个 CNN 本体嵌入模型来读取 POI name 数据。划分时间段和城市网格来 embedding 时间信息。

  2. 时空偏好建模
    神经协同过滤技术。

  3. Transfer Learning

    使用了两个矩阵来交互使用不同城市间的信息，决定使用多少量。
